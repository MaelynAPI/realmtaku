# Realmtaku

Realmtaku adalah sebuah website untuk membaca komik, menonton anime, dan donghua secara online.  
Website ini menggunakan API dari **Omni Maelyn** untuk mengambil data komik.  

## Fitur
- **Baca Komik**: Menyediakan daftar komik terbaru berdasarkan genre dan pencarian.
- **Cari Komik**: Fitur pencarian untuk menemukan komik favorit.
- **Genre Komik**: Menampilkan daftar komik berdasarkan kategori/genre.
- **Bookmark**: Memiliki fitur bookmark
- **Auto Scroll**: Sudah dilengkapi dengan auto scroll
- **Disqus**: Sudah dilengkapi dengan disqus

## Teknologi yang Digunakan
- **Node.js** + **Express.js** sebagai backend.
- **Axios** untuk mengambil data dari API.
- **EJS** sebagai template engine untuk tampilan.

## API yang Digunakan
- **Komik**: `https://apiomni.maelyn.tech/api/komikcast`

## Cara Menjalankan
1. Clone repository ini.
2. Install dependencies:  
   ```
   npm install
   ```
3. Jalankan server:  
   ```
   npm start
   ```
4. Akses website di browser melalui `http://localhost:3110`.
